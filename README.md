# Colour2Carbon: Soil Organic Carbon (SOC) Estimation Framework
Course Project - Soil Technology

**Overview**
A non-destructive machine learning pipeline designed to estimate Soil Organic Carbon (SOC) by leveraging the inverse correlation between RGB spectral values and soil carbon density. The framework automates the generation of Soil Health Report Cards to optimize agricultural fertility and water retention planning.

**Tech Stack:** Python, Scikit-learn, Pandas, Seaborn, Jupyter

**Model Performance & Features:**
* **Algorithm:** Random Forest Regressor
* **Dataset Size:** 60,000+ diverse spectral samples
* **Performance Metrics:** R = 0.878, RMSE = 0.355
* **Output:** Automated, rule-based Soil Health Report Card classification.
